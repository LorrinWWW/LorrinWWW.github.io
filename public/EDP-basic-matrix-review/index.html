Cholesky分解

条件：

  1. une matrice hermitienne：矩阵中的元素共轭对称（复数域的定义，类比于实数对称矩阵）。Hermitiank意味着对于任意向量x和y，(x*)Ay共轭相等
  2. Positive-definite：正定矩阵A意味着，对于任何向量x，(x^T)Ax总是大于零(复数域是(x*)Ax>0)

则存在L为下三角矩阵，使得 A = LL*。

目标：A = QR，Q是正交矩阵（意味着QTQ = I）而R是上三角矩阵。
